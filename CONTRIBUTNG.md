# Contributing to CSAT

Thank you for your interest in CSAT — Construction Safety Analytics Tool!

This project began as my MSCISBA capstone at West Texas A&M University (Dec 2025), with the goal of using AI and analytics to prevent construction injuries. I'm thrilled if you'd like to use, extend, or improve it.

The only requirement is preserving the copyright notice and license text.

See the [LICENSE](LICENSE) file for details.

## How to Contribute

1. **Fork the repository** to your own GitHub account
2. Create a new branch for your changes
3. Make your improvements (new features, bug fixes, better data handling, UI enhancements, etc.)
4. Test locally with Docker
5. Submit a Pull Request

All contributions are welcome — big or small!

## I Kindly Ask

While the MIT license doesn't require it, I'd greatly appreciate if you:
- **Give credit** by keeping my name and the original repo link in any derivative work
- **Let me know** you're using or extending CSAT! Open an issue here or email me at [jkinter@gmail.com]. I'd love to hear how it's helping and feature your work.

Your improvements could literally save lives.

Thank you for helping make construction sites safer.
